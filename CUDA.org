* Nvidia modprobe
Even the CUDA driver and torch is good installed, still print(f"CUDA Available:  {torch.cuda.is_available()}") may return False, also arise problem for other cases.
Please set the modprobe and reboot:
It manually creates the Unified Memory (UVM) device files that PyTorch relies on.

#+begin_src 
  sudo nvidia-modprobe -u -c=0
  sudo nvidia-modprobe -m
#+end_src
