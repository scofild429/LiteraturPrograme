Shortcut

* Input
- Ctrl   + Tab: switch input methods


* Screen
- Option + Ctrl + Left : show page to left screen side
- Option + Ctrl + Right : show page to right screen side

- Command + + : zoom in
- Command + - : zoom out

- Command + Q : force exit

- Command + Tab : switch between different application
- Command + Â´ : swtich between the same application 


* ITerm2  
- Command + T : open new terminal tab
- Command + (number): go to corresponding tab

- Command + D : open new pan vertical
- Command + Shift + D : open new pan horizontal
- Command + Option + direction: move between pan
  
  
* Dictionary
- Command + Ctrl + D : open dictionary
- Option + ESC : pronounce the selected words
